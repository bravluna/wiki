字符实体

要在 HTML 文档里插入 < 符号需要这样写：&lt;，&lt; 叫做字符实体。程序员会疑问，为什么不用 \< 转义来插入字符？字符实体是什么？

事实上，在 HTML 文档里可以插入 Unicode 字符集中的任意字符，比如要插入 ♥，可以这样写：&#9829;，中间的４个数字是待插入字符的 Unicode 内码。

另外，为了便于记忆，可以用名字的形式，用 &hearts; 来插入 ♥，HTML 预定义了很多名字，如 &lt; 代表 <。

通过这种方式可以用多种语言书写 HTML 文档，当然并不需要都写成字符实体的形式，键盘上有的字符直接插入即可。

HTML 使用 < > & 来定义标签和插入字符实体，要在文本里使用这些字符不能直接书写。

但也不需要另外规定转义的形式，写它们的字符实体即可。

所以在 HTML 文档里字符实体有两个作用：一是插入字符，二是对 < > & 进行转义。

这里有一个专门介绍字符实体的网站，按类别提供了常用字符实体参考。
